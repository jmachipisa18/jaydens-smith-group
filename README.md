# Jayden's Smith Group

Production-style React + Vite website for a 24/7 emergency electrician and plumbing company in Leicester.

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- React Router
- Lucide React
- Framer Motion
- ESLint
- Prettier

## Installed Tooling Check

`node` and `npm` were not installed globally on this machine, so a local Node 20 runtime was added inside this workspace at `node-v20.19.0-darwin-arm64/`.

## Run Locally

Start the dev server:

```bash
PATH="/Users/topstriker/Visual Code/node-v20.19.0-darwin-arm64/bin:$PATH" npm run dev
```

Build for production:

```bash
PATH="/Users/topstriker/Visual Code/node-v20.19.0-darwin-arm64/bin:$PATH" npm run build
```

Preview the production build:

```bash
PATH="/Users/topstriker/Visual Code/node-v20.19.0-darwin-arm64/bin:$PATH" npm run preview
```

Lint the project:

```bash
PATH="/Users/topstriker/Visual Code/node-v20.19.0-darwin-arm64/bin:$PATH" npm run lint
```
